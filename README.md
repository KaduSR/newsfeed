# NewsFeed 📰

> Agregador de noticias com 5 fontes customizaveis e funcionamento 100% offline.

## Stack
React • PWA • IndexedDB • Service Worker • News API

## Funcionalidades
- 5 fontes de noticias customizaveis
- Busca full-text com highlight
- Favoritos offline (IndexedDB)
- PWA instalavel na tela inicial
- Cache inteligente com Service Worker

## Metrica
100% funcional offline — acesso a noticias sem internet.
